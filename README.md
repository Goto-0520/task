## 使用方法

- 以下の手順でプロジェクトをローカル環境にインストールしてください。


リポジトリをクローン
```bash
git clone 
```

ディレクトリに移動
```bash
cd task
```

依存関係をインストール
```bash
chmod +x calculation
```

実行方法

```bash

./calculation

```
