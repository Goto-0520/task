## テスト環境
Ubuntu 20.04以降

## 概要

四則演算に三角関が加わったもの

## 使用方法

以下の手順でプロジェクトをローカル環境にインストールしてください。


リポジトリをクローン
```bash
git clone 
```

ディレクトリに移動
```bash
cd task
```

依存関係をインストール
```bash
chmod +x calculation
```

実行方法

```bash

./calculation

```
